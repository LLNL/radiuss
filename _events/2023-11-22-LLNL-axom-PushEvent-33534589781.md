<small>FlatMap: avoid copies of bucket/metadata arrays during swap()

This fixes an issue with crashing rehash() operations for a map of
std::string values. std::string is not trivially-relocatable in
libstdc++ debug mode, so a bitcopy of the underlying object will not be
valid.</small>
